# Ret-Sales-DW
This repository hosts code, workflows, and documentation for a Retail Sales Data Warehouse project — a system designed to store, process, and organize retail sales data for reporting, analysis, and business intelligence. 
# Retail Sales & Profitability Data Warehouse
**Project Code**: `RET-SALES-DW`  
**Version**: `1.0.0`  
**Status**: Production Ready

## 📌 Overview
End-to-end data platform to analyze sales, profitability, and performance for a US supermarket chain. Built to solve inconsistent performance, identify drivers of success/failure, and enable data-driven decisions.

## 🏗️ Architecture
6-Layer Architecture:
1. Staging → 2. Integration → 3. EDW → 4. Data Marts → 5. Sandbox → 6. Presentation

## 📊 Dataset
- Source: `supermarket_064719.xlsx`
- Period: 2014–2017
- Regions: East, West, Central, South
- Segments: Consumer, Corporate, Home Office
- Categories: Furniture, Office Supplies, Technology

## 🚀 Key Deliverables
- Data Warehouse & Schema
- ETL Pipelines
- SQL/Python Analysis
- Power BI Dashboards
- KPIs & DAX Measures
- A/B Testing Framework
